# 优化主页导航与语言主题控件

## 背景

本次维护先处理主页顶部栏目和首屏入口的展示习惯，让导航更贴近个人项目、履历与阅读笔记网站的定位。

## 实施内容

- 从顶部导航和首页首屏入口移除 `Blog`。
- 将 `Papers` 导航入口改为 `Reading Notes`，并为中文 UI 模式提供 `阅读笔记` 文案。
- 将 `Theme` 文本按钮改为太阳/月亮图标按钮。
- 增加 `中/英` 语言切换按钮，只翻译显式标记的导航、首页入口、栏目卡片和页脚文案。
- 将 `.tmp/芙莉莲.png` 复制为正式静态资源 `static/img/frieren.png`，用于白天模式主页头像；夜间模式继续使用原头像。
- 为 `GitHub` 等外部链接增加跳转图标和下划线提示。

## 决策

- 语言切换采用 `data-i18n` 显式标记机制，避免误改论文笔记正文。
- `.tmp/` 继续只作为本地临时目录，网页引用的图片资源必须进入 `static/img/`。
- `content/posts/` 暂不删除，只从可见主页和顶部导航入口移除。


构建和生产产物检查均通过。
