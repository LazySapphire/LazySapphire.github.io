# 新增 AI 技能选择引导

日期: 2026-06-03

## 背景

后续仓库可能添加多个 skill。不是每个任务都需要加载所有 skill，因此需要一个轻量入口来帮助新的 AI agent 判断任务类型，并只加载匹配的 skill。

## 本次更新

- 新增 `.ai/AGENTS.md` 作为 AI agent 进入仓库后的技能选择入口。
- 明确不将“选择 skill”本身做成 skill，避免递归式依赖。
- 在入口文档中记录当前 skill:
  - `.ai/skills/repository-maintenance-cycle/SKILL.md`
- 记录 `.tmp/`、`.maintenance/logs/`、`public/`、源码目录的职责。
- 规定新增 skill 时需要同步更新 `.ai/AGENTS.md` 并写维护日志。

## 验证

- 文档位于 `.ai/` 下，不在 Hugo 发布目录 `content/` 或 `static/` 内。
- 不会进入 `public/`。

## 后续

- 每新增一个 skill，都应更新 `.ai/AGENTS.md` 的 Current Skills 表。
