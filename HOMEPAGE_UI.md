## 2. 导航栏 (Navigation Bar)

### 2.1 桌面端导航

| 元素 | 类型 | 说明 | 链接/功能 |
|------|------|------|-----------|
| **HOMEPAGE** | Logo/文字链接 | 网站首页链接 | `/` |
| **Blog** | 下拉菜单 | 博客菜单，带下拉选项 | `/posts/` |
| └ example sub-menu | 子菜单项 | Google图标示例 | `/posts/` |
| **Projects (项目)** | 下拉菜单 | 项目展示菜单 | `#` |
| └ Offline RL (离线强化) | 子菜单项 | 强化学习项目 | `#` |
| └ Robotics (机器人) | 子菜单项 | 机器人项目 | `#` |
| **Tools (工具)** | 下拉菜单 | 工具资源菜单 | `#` |
| └ Dev Tools (开发工具) | 子菜单项 | 开发工具集合 | `#` |
| └ Utils (实用工具) | 子菜单项 | 实用工具集合 | `#` |
| **Resume (简历)** | 文字链接 | 个人简历页面 | `/resume/` |
| **GitHub (代码)** | 图标+文字链接 | 外部GitHub链接 | `https://github.com/LazySapphire` (新标签页) |
| **🔍 搜索按钮** | 图标按钮 | 打开搜索弹窗 | 触发搜索功能 |
| **中/EN** | 文字按钮 | 中英文切换 | 切换导航按钮语言 |
| **🌙/☀️ 主题切换** | 图标按钮 | 深色/浅色模式切换 | 切换页面主题 |

### 2.2 移动端导航

| 元素 | 类型 | 说明 |
|------|------|------|
| **汉堡菜单按钮 (☰)** | 图标按钮 | 展开/收起移动端菜单 |

### 2.3 移动端菜单展开内容

- Blog (可展开/收起子菜单)

- Tools (可展开/收起子菜单)

## 6. 搜索弹窗 (Search Modal)

### 6.1 触发方式
- 点击导航栏的搜索按钮
- 快捷键 (通常支持 `/` 或 `Ctrl+K`)

### 6.2 弹窗结构

```
┌─────────────────────────────────────┐
│ 🔍  [搜索输入框...]              ✕ │  ← 头部 (Header)
├─────────────────────────────────────┤
│                                     │
│  [搜索结果列表]                     │  ← 结果区域
│                                     │
└─────────────────────────────────────┘
```

### 6.3 功能说明
- **占位符**: "Search"
- **关闭按钮**: ✕ 图标，悬停变主色
- **背景遮罩**: 半透明 + 模糊效果
- **快捷键关闭**: Esc 键

---

## 7. 功能按钮详解

### 7.1 主题切换按钮
- **深色模式图标**: 🌙 (月亮)
- **浅色模式图标**: ☀️ (太阳)
- **切换逻辑**: 点击切换 dark/light class
- **存储**: 使用 localStorage 记住用户选择

### 7.2 语言切换按钮
- **显示文字**: "EN" 或 "中"
- **位置**: 搜索按钮和主题切换按钮之间
- **切换逻辑**: 点击在中英文之间切换
- **作用范围**: 仅切换导航栏按钮文字，不改变页面内容语言
- **存储**: 使用 localStorage 记住用户语言偏好

|------|------|

### 7.3 搜索按钮
- **图标**: 🔍 (放大镜)
- **位置**: 桌面端导航栏右侧，移动端汉堡菜单旁
- **行为**: 打开全屏搜索弹窗

### 7.4 汉堡菜单按钮
- **图标**: ☰ (三条横线)
- **位置**: 移动端导航栏最右侧
- **行为**: 展开/收起全屏菜单

## 8. 无障碍功能

| 功能 | 说明 |
|------|------|
| **Skip to main content** | 页面顶部隐藏的跳转链接，方便键盘用户跳过导航 |
| **aria-label** | 所有图标按钮都有描述性标签 |
| **焦点样式** | 支持键盘导航的焦点高亮 |

---

## 9. 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| **默认** | < 640px | 单列布局，汉堡菜单 |
| **sm** | ≥ 640px | 2列文章网格 |
| **md** | ≥ 768px | 3列文章网格，显示完整导航 |
| **lg** | ≥ 1024px | 更大间距 |
| **xl** | ≥ 1280px | 最大宽度限制 |
